# CP-OSINT v2

**CP-OSINT v2** is an educational OSINT (Open-Source Intelligence) and network forensic analysis tool designed for learning, research, and defensive security analysis.

This project focuses on **network footprint analysis**, **domain/IP intelligence**, and **forensic-style reporting**, without performing exploitation or intrusive attacks.
